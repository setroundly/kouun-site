# kouun-site

会社公式サイト（静的HTML）。

## 構成

```
kouun-site/
├── index.html      # トップページ
├── css/
│   └── style.css
├── images/         # 画像置き場
└── README.md
```

## ローカル確認

`index.html` をブラウザで開くだけで確認できます。

## FTPアップロード

このフォルダの中身を、サーバーの公開ディレクトリ（`public_html` など）にそのままアップロードしてください。

- `index.html` がドキュメントルート直下にあること
- `css/` フォルダも一緒にアップロードすること

## 会社PCへの移行

`kouun-site` フォルダごとコピーすれば、そのまま編集・FTPアップロードが可能です。
